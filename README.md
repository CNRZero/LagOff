# LagOff - Gaming Performance Optimizer

A modern, responsive landing page for LagOff, a gaming performance optimization tool.

## Features

- **Modern Design**: Clean, dark-themed UI with gradient accents
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-triggered animations, parallax effects, and micro-interactions
- **Optimized Performance**: Efficient JavaScript with intersection observers and debounced scroll events

## Sections

1. **Hero** - Eye-catching introduction with animated stats and live dashboard preview
2. **How It Works** - 4-step process explaining the tool's simplicity
3. **Features** - 6 key features with hover effects
4. **Testimonials** - User reviews with tilt effects
5. **About** - Benefits and reasons to choose LagOff
6. **CTA** - Call-to-action section for conversions
7. **Footer** - Links, social media, and legal information

## Technologies

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Getting Started

Simply open `index.html` in a modern web browser.

```bash
# Or serve locally
python -m http.server 8000
# Visit http://localhost:8000
```
